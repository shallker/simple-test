simple-test
==========

Simple test


## Installation
```bash
npm install simple-test
```

## Quick Start
```javascript
var test = require('simple-test');

test('add user', function (done) {
  done();
});
```

## API

### test(String name, Function fn)
#### .async(String name, Function fn)
